⚡ SETTINGS
Google Sheets Webhook URL

Paste your Google Apps Script Web App URL here to automatically send orders to your Google Sheet. See setup instructions below.

Google Sheets Setup (10 min setup)
1. Go to sheets.google.com → create a new sheet
2. Name row 1 headers: OrderID | Name | Room | Phone | Slot | Items | Total | Notes | Time
3. Click Extensions → Apps Script
4. Paste the script below → Save → Deploy → Web App
5. Set "Who has access" to Anyone → Copy URL → Paste above
function doPost(e){
  var data = JSON.parse(e.postData.contents);
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var items = data.items.map(i => i.qty+'x '+i.name+' ('+i.vendor+')').join(', ');
  sheet.appendRow([
    data.id, data.name, data.room, data.phone,
    data.slot, items, '₹'+data.total, data.notes,
    new Date(data.timestamp).toLocaleString('en-IN')
  ]);
  return ContentService.createTextOutput('OK');
}
